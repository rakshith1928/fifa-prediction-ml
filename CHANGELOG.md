# Changelog

## [1.0.0] - 2025

### Added

- Historical match data processing (1872-present)
- ELO rating system implementation
- Rolling team form features (last 5 matches)
- Head-to-head statistics
- XGBoost classifier for match prediction
- England vs. Croatia prediction pipeline
- Feature importance analysis
